📘 Assignment 1 – HTML Tags Demonstration
📌 Overview

This project is a simple HTML webpage that demonstrates the usage of basic HTML tags. It presents information in a structured table format, making it easy to understand common tags and their purposes.

🎯 Objective
To understand fundamental HTML tags
To practice creating tables in HTML
To display tag descriptions with examples
🛠️ Technologies Used
HTML5
📂 Project Structure
ass1/
│── index.html   # Main HTML file containing the assignment
🚀 Features
Displays a table of commonly used HTML tags
Includes tag name, description, and example
Uses basic styling with attributes like bgcolor, border, and cellpadding
Demonstrates tags such as:
<b> (Bold)
<i> (Italic)
<u> (Underline)
<br> (Line Break)
<marquee> (Scrolling text)
<a> (Hyperlink)
<table> (Table structure)
💡 How to Run
Download or clone the repository:
git clone https://github.com/TEJAS-PARKAR/FSD.git
Navigate to the assignment folder:
cd FSD/ass1
Open the file:
Double-click index.html
OR
Right-click → Open with browser
📸 Output
A yellow background webpage
A green table displaying HTML tag information
A scrolling text ("LOL!!!") using the <marquee> tag
⚠️ Notes
Some tags like <marquee> are deprecated in modern HTML and should not be used in production projects
Inline attributes like bgcolor are outdated; CSS is recommended instead
