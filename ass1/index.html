Information about HTML tags.

TAG DESCRIPTION EXAMPLE

Caption Tag

 This tag is used to add a caption under tables, photos, etc. <caption>This is a photo</caption>
Bold Tag Used to write something in BOLD <b>THIS IS BOLD</b>
Italics Tag Used to write something in italics <i>This is in ITALICS</i>
Underline Tag This tag is used to underline a certain text <u>This is UNDERLINED TEXT</u>
Break Tag This tag is used break line and go to new line <br>This is <br>BREAK TAG
Marquee Tag This tag is used if we want a certain text rotating horizontally throughout our screen <marquee>LOL!!</marquee>
Main Tag Used to write something as the main content <main>THIS IS THE MAIN CONTENT</main>
Paragraph Tag Used to write something as a paragraph <p>This is a paragraph</p>
Anchor Tag Used to insert a hyperlink <a href="www.google.com">GOOGLE</a>
Table Tag Used to make a table in the webpage <table><tr><th>SINGLE ROW AND COLUMN</th></tr></table>
Simple HTML tags.
