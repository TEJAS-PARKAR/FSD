🌐 Assignment No.-2 Personal Portfolio Website – Tejas Parkar
📌 Overview

This is a personal portfolio website showcasing my skills, projects, experience, and achievements as a Computer Engineering student.
The website is fully built using frontend technologies and is designed to provide a structured and interactive overview of my profile.

🎯 Objective
Present personal and professional information in a clean UI
Highlight technical skills and projects
Provide an easy way for recruiters and peers to connect
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling (via styles.css)

│── styles.css       # Styling file
